# **App Name**: Ankur's Echo Chamber

## Core Features:

- AI/ML Showcase: Display Ankur's AI/ML projects, including the Termux CLI AI agent (code, description, and potential demo).
- Vocal Journey: Showcase Ankur's musical practice journey, favorite songs, and influences (A.R. Rahman, Zubeen Garg, JB, The Weeknd, Arctic Monkeys).
- About Me: A dedicated section detailing Ankur's background, age, interests, and aspirations. Include the AI/ML career goals.
- Interactive Chatbot: Allow site visitors to chat with a personalized AI Chatbot highlighting the ability to write/read/analyze/plan/orchestrate files. Use it to showcase your chatbot building expertise.

## Style Guidelines:

- Primary color: Deep midnight blue (#192A56) to set a dark and sophisticated tone.
- Background color: Near-black (#0A0E17), subtly lighter than true black to provide depth.
- Accent color: Electric blue (#7DF9FF) for interactive elements, reflecting a futuristic aesthetic.
- Font pairing: 'Space Grotesk' for headlines and 'Inter' for body text.
- Use simple, glowing icons, designed in the accent color, to represent different sections and functionalities.
- Implement a responsive grid layout for compatibility across devices (mobile and desktop).
- CSS-drawn stars with subtle animations twinkling in the background. Smooth transitions for button clicks and hover effects that cause a soft glow.
